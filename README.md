## 📖 Project Overview

This project is an **End-to-End Data Analytics and Machine Learning Capstone Project** completed during the **Syntecxhub Internship (Task 4)**.

The objective of this project is to analyze employee salary data, identify important factors influencing salary, perform exploratory data analysis (EDA), and build a machine learning model capable of predicting employee salary categories.

---

## 🎯 Objectives

- Clean and preprocess real-world employee salary data
- Perform Exploratory Data Analysis (EDA)
- Visualize important trends and patterns
- Build a Machine Learning classification model
- Evaluate model performance
- Generate business insights for decision-making

## 📈 Key Insights

- Most employees belong to the **<=50K** salary category.
- Employees with higher education levels tend to earn higher salaries.
- Capital Gain is one of the strongest indicators of salary.
- Employees working more hours per week are more likely to have higher incomes.
- Occupation and Education significantly influence salary prediction.
